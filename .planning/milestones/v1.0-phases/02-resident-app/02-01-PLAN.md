## Task 1: Create ResidentHeader component

Create `mobile/components/ResidentHeader.tsx`:

- Fixed position top bar, height 64px, z-index 50
- Background: `rgba(9,14,24,0.6)` with BlurView (intensity 40, tint dark)
- Bottom border: `1px solid rgba(148,163,184,0.1)`
- Left side: 40×40 avatar placeholder (rounded-full, bg `#25293A`), name "Aarav Mehta", unit "A-402"
- Right side: Notification bell icon (`notifications-none` from MaterialIcons, color `#9BABCE`), with animated yellow ping dot (backgroundColor `#FACC15`, pulsing opacity animation via `Animated.loop`)
- Use Inter font family throughout
- Safe area aware (use `useSafeAreaInsets` from `react-native-safe-area-context`)

### Verification
